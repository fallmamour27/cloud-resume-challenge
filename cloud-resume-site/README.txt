# Cloud Resume Challenge 🌥️

This project is part of the [Cloud Resume Challenge](https://cloudresumechallenge.dev/), a hands-on initiative to build and deploy a professional resume using cloud technologies.

## 🧠 Skills Demonstrated
- HTML5 + CSS3 (based on the Solid State template by HTML5 UP)
- Git & GitHub for version control
- AWS (S3, CloudFront, Route 53) [planned]
